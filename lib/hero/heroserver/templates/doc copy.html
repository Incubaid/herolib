Authentication Required

All API endpoints require authentication using a session key obtained through the authentication flow:

  1. Register your public key: POST /auth/register
  2. Request authentication challenge: POST /auth/authreq
  3. Sign challenge and submit: POST /auth/auth
  4. Use returned session key in Authorization: Bearer {session_key} header
