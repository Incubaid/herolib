Quick Start

Getting Handler Information

Get detailed information about a specific API handler in JSON format:

curl ${server_info.base_url}/json/[handler_name]
Making API Calls

All API endpoints use JSON-RPC 2.0 format. Here's a basic example:

curl -X POST ${server_info.base_url}/api/[handler_name] \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "method_name",
    "params": {
      "param1": "value1",
      "param2": "value2"
    },
    "id": 1
  }'
Tip: Visit the documentation pages for specific examples with copy-to-clipboard functionality
