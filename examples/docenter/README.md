# Docenter

A markdown editor with WebDAV backend, built with V.

## Quick Start

```bash
./examples/docenter/docenter_server.vsh
```

Open your browser to `http://localhost:8004`

Default credentials: `admin` / `123`

## Features

- Markdown editor with live preview
- WebDAV file storage
- Collection-based organization
- Database-backed persistence
